A brief history of Linux

  • Linux was first devloped by Linus Torvold in 1991 as a free distributable version of the Unix operating system.
  • The first official versions of linux was version .02 was released on october 5, 1991. This version was very simple with nothing more than the ability to run bash and gcc.
  • As more people started to contribute to the project and development went along the first version of linux was made publicly available in March of 1994.
  • Today Linux is now considered a complete Unix clone with a graphical interface capable of web surfing, mail software and word processing applications. This on top of the many commercial programs that have been ported over.
  • As of September 10, 2019 the latest stable linux kernel is 5.2.13.

Linux as an operating system

  • One of Linux's greatest draws is it being open-source which allows anybody to look into every bit of the system and modify them as they want.
  • Linux as an operating is maintaned by people taking the time to contribute to the code in attempts to improve upon the language.
  • Due to it being open-source there is a sense of safety and privacy due to any malicious code that can be inserted is open for the world to see.
  • Due to the afromentioned security and widespread support either from users or commercial companies has made linux very popular as an operating system for web-servers.
  • Today, linux servers generate more than one-third of all web-pages on the internet.
